#"Система для управления рецептами"

Я разработала систему для управления рецептами. Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.


Код можно запустить через терминал или через среду разработки, например Pycharm.

#Установка

```bash
git clone https://github.com/anfisaoparina/HW2TP.git
cd HW2TP
pip install -r requirements.txt
pytest
```


#Как запустить?

В терминале: 
```bash
cd HW2TP
python3 task1.1.py
```
*команда python3 для мака, а для виндоус просто можно python
